
# BMI Calculator with Categories

## What it does
Calculates Body Mass Index and tells you your category.

## Formula
BMI = weight(kg) / (height(m)²)

## How to use
1. Enter height in centimeters
2. Enter weight in kilograms
3. Click "Calculate BMI"
4. See result and category

## Categories
- Underweight: < 18.5
- Normal weight: 18.5 - 24.9
- Overweight: 25 - 29.9
- Obese: ≥ 30
